# 📊 Calculadora de IMC com Streamlit

Aplicação web desenvolvida em Python para cálculo do IMC (Índice de Massa Corporal), com interface interativa usando Streamlit.

## 🚀 Funcionalidades
- Cálculo automático do IMC
- Classificação do resultado (baixo peso, normal, sobrepeso, obesidade)
- Interface simples e interativa no navegador

## 🛠️ Tecnologias
- Python
- Streamlit

## ▶️ Como executar

### 📋 Pré-requisitos
- Python 3.8 ou superior instalado

---

### 🚀 Passo a passo

#### 1. Baixar o projeto
Baixe o repositório (ZIP) ou clone com Git:

```bash
git clone https://github.com/Joao-VMD/streamlit-imc.git
```

---

#### 2. Acessar a pasta

```bash
cd streamlit-imc
```

---

#### 3. Instalar dependências

```bash
pip install streamlit
```

---

#### 4. Executar a aplicação

```bash
streamlit run app.py
```

---

### 🌐 Acessar no navegador

http://localhost:8501

---

### 🛑 Para parar

Pressione Ctrl + C no terminal
